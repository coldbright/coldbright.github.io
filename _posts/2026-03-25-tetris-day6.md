# 🎮 테트리스 리뉴얼 개발 일지 - 6일차

오늘은 대기방에서 발생하던 버그를 수정하고 게임 화면 연결 기능을 구현하였다.

---

## 📌 오늘의 개발 목표

* 대기방 버그 수정
* 게임 시작 기능 구현

---

## 🛠 구현 내용

### 플레이어 이름 표시 오류 수정

대기방에서 상대방의 이름이 표시되지 않는 문제를 발견하였다.

원인을 분석한 결과 데이터 전달 과정에서 일부 정보가 누락되고 있었으며 이를 수정하였다.

### 게임 화면 연결

모든 플레이어가 준비 상태가 되면 게임 화면으로 이동하도록 구현하였다.

### 사용자 흐름 개선

로비 → 대기방 → 게임 화면

과정이 자연스럽게 이어지도록 로직을 수정하였다.
