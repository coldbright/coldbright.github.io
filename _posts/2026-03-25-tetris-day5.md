# 🎮 테트리스 리뉴얼 개발 일지 - 5일차

---

## 📌 오늘의 개발 목표

* 대기방 UI 개선
* 로그인 시스템 개편
* 데이터베이스 구조 개선

---

## 🛠 구현 내용

### 대기방 UI 개선

로비 디자인과 통일감을 주기 위해 대기방 화면도 새롭게 디자인하였다.

### 로그인 시스템 개편

UI 변경에 따라 기존 로그인 시스템을 수정하고 추가 기능을 지원하도록 개선하였다.

### 데이터베이스 정규화

기존에는 모든 정보가 `user` 테이블에 저장되어 있었다.

이를 정규화를 통해 다음과 같이 분리하였다.

```text
user
└─ 계정 정보

user_stats
└─ 전적 및 통계 정보
```

### 정규화 효과

* 데이터 중복 감소
* 관리 효율 향상
* 확장성 확보
