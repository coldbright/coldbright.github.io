# 🎮 테트리스 웹게임 개발 일지 - 2일차

오늘부터 **웹 브라우저에서 실행되는 테트리스 게임**을 개발하기로 했다.
이번 프로젝트의 목표는 **HTML, CSS, JavaScript, Node.js를 활용하여 멀티 플레이어 테트리스 게임을 구현하는 것**이다.

---

## 📌 프로젝트 목표

이번 프로젝트에서 구현하고 싶은 기능은 다음과 같다.

### 게임 플레이 측면
- 테트리스 블록 생성
- 블록 이동 및 회전
- 블록 충돌 감지
- 줄 완성 시 제거
- 점수 시스템
- 게임 오버 기능

### 멀티 플레이 측면
- 대기방 구현
- 상태창(status) 구현
- 준비완료 구현
- 줄 제거시 상대방의 체력 감소 기능

최종적으로는 **웹 브라우저에서 바로 플레이할 수 있는 멀티 테트리스 게임**을 만드는 것이 목표다.

---

## 🛠 사용 기술

이번 프로젝트에서 사용할 기술 스택은 다음과 같다.

- **HTML** : 게임 화면 구조
- **CSS** : 게임 UI 디자인
- **JavaScript** : 게임 로직 구현
- **Canvas API** : 게임 화면 렌더링
- **Node.js(express, 등)** : 게임 백엔드 실행

---

## 🧩 테트리스 기본 구조 설계

테트리스 게임은 다음과 같은 구조로 구성된다.

3. **블록 자동 낙하**
4. **충돌 감지**
