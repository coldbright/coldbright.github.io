## 🛠 구현 내용

### 서버 오류 수정

기존 서버에서 발생하던 다양한 예외 상황과 데이터 처리 오류를 수정하였다.

### 서버 구조 개선

기존에는 모든 서버 코드가 `server.js` 하나에 작성되어 있어 코드 관리가 어려웠다.

이를 해결하기 위해 기능별로 파일을 분리하였다.

```bash
project-root/
├── config/
│   └── db.js
├── routes/
│   ├── stats.js
│   └── user.js
└── server.js
```

### 분리된 역할

* `db.js` : 데이터베이스 연결 관리
* `user.js` : 사용자 관련 API
* `stats.js` : 전적 및 통계 API
* `server.js` : 서버 실행 및 라우터 연결
